# AwesomePry

Wrapper gem to wrap some useful gems around pry - `pry-byebug`, `awesome_print`, `pry-doc`, `pry-stack_explorer`

## Installation

Add this line to your application's Gemfile:

    gem 'awesome-pry'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install awesome-pry
